Contributing to Microprobe¶

All pull requests submitted must have their commits signed. You can have git do this automatically with git commit -s. Alternatively, add a line like the following to the end of your commit:

Signed-off-by: Your Name <your.name@your.email>

When submitting patches/pull requests, you agree to the following Developer Certificate of Origin, i.e., items (a–d):
